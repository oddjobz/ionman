This is template is taken directly from the Bootstrap examples, just as an example of what can be shown on a web page. The real demonstration is the framework behind the page, not that one ach page transition (when you click on the green button) the page itself is not refreshed, but a "part" of the page IS. That leaves all your CSS/JS (and websockets) in place over transitions which makes it MUCH faster than loading a new page .. AND much smoother.

Note also that each page is a JINJA2 template as you would find served up by Flask (although the page is actually being served over a websocket, this template formatting is also used by Ansible ..
